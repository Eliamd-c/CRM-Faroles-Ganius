Asistente de Webhook para Desarrolladores

Para recibir mensajes reales, Meta debe enviar eventos a tu servidor a través de HTTPS. Sigue estos pasos:

  1. Corre una herramienta de túnel como ngrok localmente: 
    ngrok http 5000
  2. Copia la dirección HTTPS externa provista por ngrok (ej: https://abcdef.ngrok-free.app).
  3. En el panel de tu Meta App -> Webhooks de Instagram, ingresa esa URL agregándole /webhook al final. 
    https://abcdef.ngrok-free.app/webhook
  4. Configura el Webhook Verify Token de arriba en ambos lados.
