# Reglas del Proyecto (CMR Faroles)

- **Despliegue Automático (CI/CD)**: Todo cambio realizado en los archivos del código fuente del proyecto (ej: plantillas HTML, CSS, JavaScript, backend de Python, dependencias, etc.) debe ser confirmado (commit) y subido (push) de forma automática al repositorio remoto en GitHub (`https://github.com/Eliamd-c/CRM-Faroles-Ganius.git`) para activar el despliegue automático del servicio en la nube (Render.com/Railway).
